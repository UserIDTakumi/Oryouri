# Oryouri
出汁ご飯<br>
  【注記】<br>
    ・炊飯器を使います<br>
    ・出汁ご飯と言いつつも出汁は取りません(←???)<br>
<br>
  【材料】<br>
    ・米<br>
    ・ヒガシマル うどんスープ<br>
  【レシピ】<br>
    1）米を研ぐ(無洗米なら不要)<br>
    2）米を浸漬する(夏場は1時間、冬場は1~2時間,一部炊飯器のモデルによっては不要な場合も)<br>
    3）米を炊飯釜に入れ、ヒガシマル　うどんスープを1合に対し、1袋(8g)を入れる<br>
    4）炊飯器の規定通りの水量にし、"炊き込みご飯"モードで炊飯する<br>
    5）完成<br>
